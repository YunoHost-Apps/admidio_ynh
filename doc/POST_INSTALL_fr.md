### Post-installation
Se rendre à l'adresse <https://__DOMAIN____PATH__> pour créer l'organisation, l'utilisateur admin et renseigner les champs pour la base de données avec les données suivantes :   
> Database system: Mariadb  
  Host : `localhost`  
  Database Name : `__DB_NAME__`  
  Username: `__DB_USER__`  
  Password: `__DB_PWD__`
